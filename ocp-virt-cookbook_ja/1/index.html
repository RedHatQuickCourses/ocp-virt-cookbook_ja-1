OpenShift Virtualization クックブック

はじめに

OpenShift Virtualization でワークロードを開始およびデプロイするための実用的なフィールドガイドです。

このクックブックは、OpenShift を使い始めたばかりのユーザー向けに、一口サイズのレシピ、マニフェストの例、およびトラブルシューティングのメモを提供します。これは、公式ドキュメントとトレーニングコースを実践的に補完するものとして設計されており、理論的な学習に代わるものではなく、実際の導入を促進します。

(最終更新日: {build-date})

前提条件

このクックブックを効果的に使用するには、以下が必要です。

  • クラスター: OpenShift Virtualization オペレーターがインストールされた OpenShift 4.18+。

  • ツール: oc、kubectl、および virtctl。(オプション: kustomize、helm、ansible)。

  • 知識: YAML と CLI ワークフローに精通していること (必須ではありませんが、強く推奨します)。Kubernetes/OpenShift の概念に関する基本的な理解。

特にテクノロジーを初めて使用する場合は、はじめに のすべてのモジュールを参照して、環境を準備または検証してください。

クックブックモジュール

開始するには、左メニューからモジュールを 1 つ選択してください。

  • はじめに - 必須ツールのセットアップと環境検証。

  • storage:index.adoc - 仮想マシン用のストレージセットアップ。

  • networking:index.adoc - ネットワークおよび IP 設定ガイド。

  • vm-configuration:index.adoc - VM アクセス、DNS、ノード配置、およびテンプレート。

  • performance:index.adoc - CPU ピニング、NUMA、ヒュージページ、およびリアルタイム VM。

  • vm-lifecycle:index.adoc - ゴールデンイメージ、クローン作成、スナップショット、およびブート設定。

  • YAML マニフェストライブラリ - デプロイ準備が整ったマニフェスト例のライブラリ。

  • API統合 - クローン作成、エクスポート、マシン拓​​ポロジーなどの API コンポーネントのアーキテクチャーと機能。
