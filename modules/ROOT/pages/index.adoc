= OpenShift 仮想化クックブック
:navtitle: 紹介

== 紹介

OpenShift 仮想化でワークロードを開始およびデプロイするための実践的なフィールドガイド。

このクックブックは、YAML、CLI ワークフロー（必要ではありませんが強く推奨されます）、Kubernetes/OpenShift の基本的な概念についての理解が必要です。理論的な学習を補完し、置き換えるわけではありません。実世界の実装を促進します。

(__最終更新日: {build-date}__)

== 前提条件

このクックブックを効果的に使用するには、以下のものが必要です。

* **クラスター**: OpenShift 4.18+ に OpenShift 仮想化オペレーターがインストールされている。
* **ツール**: `oc`, `kubectl`, および `virtctl`。（オプション: `kustomize`, `helm`, `ansible`）。
* **知識**: YAML についての熟練度、CLI ワークフロー（必須ではありませんが強く推奨されます）。Kubernetes/OpenShift の概念についての基本的な理解。

すべての **xref:getting-started:index.adoc[]** モジュールを確認し、特に新しい技術に触れている場合は、環境を準備または検証してください。

== クックブック モジュール

左メニューから 1 つのモジュールを選択して始めてください。

* **xref:getting-started:index.adoc[]** - 必須のツールのセットアップと環境検証。
* **xref:storage:index.adoc[]** - 仮想マシンのストレージ設定。
* **xref:networking:index.adoc[]** - ネットワークおよび IP 設定ガイド。
* **xref:vm-configuration:index.adoc[]** - VM アクセス、DNS、ノード配置、テンプレート。
* **xref:performance:index.adoc[]** - CPU ピニング、NUMA、hugepages、リアルタイム VM。
* **xref:vm-lifecycle:index.adoc[]** - ゴールデンイメージ、クローン、スナップショット、ブート構成。
* **xref:manifests:index.adoc[]** - デプロイ用の準備ができたマニフェストの例のライブラリー。
* **xref:api:index.adoc[]** - クローン、エクスポート、マシントポロジーなどの API コンポーネントのアーキテクチャと機能。
